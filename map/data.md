# Данные

Статические данные для иконок и коллекций.

## Ключевые файлы
- [allIcons.ts](src/data/allIcons.ts) - Все доступные иконки
- [iconCollections.ts](src/data/iconCollections.ts) - Коллекции иконок по категориям (фигуры, звезды, стрелки, природа, символы, бизнес, украшения, значки, медицина, транспорт, государство, гербы)
